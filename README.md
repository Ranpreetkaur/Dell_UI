# Dell_UI_Challenge

### Steps for running the app
1.  Extract the Folder
2.  Run the index.html file in Chrome with --allow-file-access-from-files flags

### Features

1.  Sorting in the basis of Rating and Reviews. One can also reset the Sorting.
2.  Searching per page on the basis of Cuisines, Name, Rating and Reviews
3.  Pagination with 45 results displayed per page.
